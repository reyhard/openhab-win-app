How should the sitemap expansion affect Main UI?

Main UI is the default center surface. The question is what happens when the user clicks the sitemap toggle.

A. Expand Window, Preserve Main UI

The window grows to the right and adds a sitemap column. Main UI keeps its working width, which best matches “possible to see both” without squeezing web content.

B. Split Existing Width
